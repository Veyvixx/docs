All commands use Discord's slash command system. Start with `/` and select the command from the menu.

## Utility

| Command | Description |
|---|---|
| `/ping` | Check bot and database latency |
| `/about` | Bot stats, server count, and uptime |
| `/avatar [user]` | View a user's server or global avatar |
| `/banner [user]` | View a user's profile banner |
| `/userinfo [user]` | Detailed member profile |
| `/serverinfo` | Server statistics |
| `/roleinfo <role>` | Role details and permissions |
| `/whois [user]` | Full profile with banner, badges, and account info |
| `/variables` | Show all available template variables |
| `/help` | Browse all commands by category |
| `/premium` | View your premium status and tier |

## Moderation

| Command | Description | Permission |
|---|---|---|
| `/ban <user>` | Permanently ban a member | Ban Members |
| `/unban <id>` | Unban a user by ID | Ban Members |
| `/kick <user>` | Kick a member | Kick Members |
| `/timeout <user> <dur>` | Apply a timeout (e.g. `10m`, `2h`, `1d`) | Moderate Members |
| `/untimeout <user>` | Remove a timeout | Moderate Members |
| `/mute <user> [dur]` | Role-based mute with optional duration | Manage Roles |
| `/unmute <user>` | Remove a role-based mute | Manage Roles |
| `/jail <user>` | Restrict to the jail channel | Manage Roles |
| `/unjail <user>` | Release from jail | Manage Roles |
| `/lock [channel]` | Lock a channel | Manage Channels |
| `/unlock [channel]` | Unlock a channel | Manage Channels |
| `/refreshchannel` | Recreate channel, clearing all messages | Manage Channels |
| `/warn <user> <reason>` | Issue a formal warning | Moderate Members |
| `/warnings <user>` | View paginated warning history | Moderate Members |
| `/clearwarnings <user>` | Clear all warnings for a member | Moderate Members |
| `/purge <amount>` | Bulk delete up to 100 messages | Manage Messages |
| `/automod spam` | Enable/disable spam filter | Manage Guild |
| `/automod invites` | Block Discord invites | Manage Guild |
| `/automod keywords` | Block custom keywords | Manage Guild |
| `/automod mentions` | Limit mass mentions | Manage Guild |
| `/automod list` | View all AutoMod rules | Manage Guild |
| `/automod disable` | Disable a rule by ID | Manage Guild |
